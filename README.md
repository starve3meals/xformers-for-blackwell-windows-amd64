# xformers Prebuilt Wheels for Blackwell

This repository provides **prebuilt xformers wheels** optimized for **Blackwell architecture** (SM 12.0) on Windows.

All wheels are compiled from source with:
- **CUDA 13.0**
- **TORCH_CUDA_ARCH_LIST=12.0**
- **Windows amd64**

### Available Builds

You can find different prebuilt versions in the [Releases](../../releases) section.
